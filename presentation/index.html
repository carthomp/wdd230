Psuedo Elements

  • Using the 'after' element, we can add the same content as a new element to the end of each line.
  • You could use it to slap a '->' after each link,
  • or just to add some fun style and flair.
  • You can format this extra text as you would any other element!

Want to do that weird thing they do in old storybooks where the first letter is huge? Use 'first-letter'
